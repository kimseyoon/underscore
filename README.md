Underscore Library 구현하기 프로젝트
========

## 개요

본 프로젝트는 자바스크립트에서 용이하게 사용할 수 있는 함수를 직접 만들어 보고자 시작된 개인 프로젝트입니다.
나름 TDD(Test-driven development)로 테스트코드를 먼저 만들고 개발을 진행할것 입니다.

## 사용한것들
1. 바닐라 JavaScript
2. Unit Test Framework - Mocha
